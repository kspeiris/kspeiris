> **Real-time user behaviour analytics & anomaly detection platform**  
> **Stack:** FastAPI, React 18, PostgreSQL, Scikit-learn (Isolation Forest), Docker Compose, WebSockets  
> **Features:** UBA engine, hybrid risk scoring, 7 database models, real-time alerts, PDF reports, RBAC.

### 2. **AI‑Driven Employee Layoff Risk Prediction**
> **Explainable AI system for workforce analytics**  
> **Stack:** Flask, Streamlit, CatBoost, TensorFlow, SHAP, LIME, SQLite  
> **Features:** Risk classification, SHAP/LIME explainability, batch prediction, interactive dashboard.

### 3. **Cyber Threat Intelligence Platform (n8n Pipeline)**
> **Automated threat intelligence orchestration**  
> **Stack:** FastAPI, Streamlit, n8n, PostgreSQL, Telegram API, NVD API  
> **Features:** Scheduled CVE collection, heuristic risk scoring, Telegram alerts, glassmorphic UI.

### 4. **ShortStay – Accommodation Marketplace**
> **High‑security full‑stack marketplace with 5 role types**  
> **Stack:** React 18, Node.js, Express, MySQL, Tailwind CSS, Chart.js  
> **Features:** Secure bookings, admin dashboards, role escalation protection, glassmorphism UI.

### 5. **QuickPOS Lite – MERN Stack POS System**
> **Production POS deployed in 50+ restaurants**  
> **Stack:** React (Vite), Node.js, Express, MongoDB, Multer  
> **Features:** Role‑based access, real‑time cart, thermal receipt printing, sales analytics.

### 7. **JFI CONSOLIDATED Stock Control System**
